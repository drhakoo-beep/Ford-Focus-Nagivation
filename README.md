# Ford Focus Navigation
Transform your old tablet or phone into a navigation system with a Ford Focus watermark!

Hello, the application requires Yandex Navigation to work; otherwise, it will not function.

First, download Yandex Navigation, then download and install Android IDE. Copy and paste the code from the files into the corresponding folders one by one. Then, build and install.
